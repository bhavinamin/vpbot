## Volume Profile Bot 

Attempting to create a cli app that fetches prices for a given ticker and calculates the latest volume profile for a given ticker. Probs starting with reading fake data in a CSV and then eventually growing to interface with an API of some sort. 

## Why? 

- To learn Rust with a relatively lightweight/simple app goal and function. 
- To learn neovim and vim motions 
- Code without without AI. To enjoy writing, learning, and solving code again. No slop, no lines of uselessness.
